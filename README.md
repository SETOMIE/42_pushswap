## Overview
PushSwap acts as an introductory project to the *basics of writing an Algorithm*. From here on out, we will approach a variety of topics such as: ```Algorithm types, efficient array-stacks sorting, data structures (and manipulation)``` as well as improving our coding efficiency by minimizing the number of operations needed to solve a problem.

It is important to note that PushSwap is not only a vital project for understanding algorithms, but it is also crucial for Technical Interview prep to showcase that you have a good problem solving skills and a good understanding of CS concepts.

Starting off with the basics of PushSwap, a.k.a, ```Algorithms and Algo-Complexity (+ measurements)```. There are many types of Algorithms, all with different functions and different measurements and can be used to complete a given task or solve specific problems. The steps given to the algorithm must be clearly defined, as they will need to be followed in a specific order to achieve the desired outcome. 

For example; ```Linear Search 'O(n)' being a highly-preferred algorithm for handling unsorted lists, small data sets, linked lists, etc. due to it's status of easy user understanding and implementation.``` It is an algo-system that grows linearly with the input's size.
